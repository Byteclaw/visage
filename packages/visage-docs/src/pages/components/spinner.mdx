# Spinner

## Examples

### Default

Spinner by default uses `currentColor` to set up stroke color and it's size is the same as current font size.

```jsx live=true
<Spinner />
```

### Custom size or color

```jsx live=true
<Spinner styles={{ color: 'blue', fontSize: 2, lineHeight: 2 }} />
```

### Use Spinner in Button

```jsx live=true
<Button>
  <Spinner styles={{ mr: 1 }} /> Button with Spinner
</Button>
```
