# Checkbox

## Import

```js transpile=false
import { Checkbox } from '@byteclaw/visage';
```

## Examples

### Default

```jsx live=true
<Fragment>
  <Checkbox defaultChecked label="Checkbox label" name="1" />
  <Checkbox label="Checkbox label" name="2" />
</Fragment>
```

### Hidden label

```jsx live=true
<Checkbox hiddenLabel label="Hidden label" name="3" />
```

### Invalid

```jsx live=true
<Checkbox invalid label="Invalid checkbox" name="invalid-checkbox" />
```

### Read only

```jsx live=true
<Checkbox readOnly label="Read only checkbox" name="4" />
```

### Disabled

```jsx live=true
<Checkbox disabled label="Disabled checkbox" name="5" />
```

### Controlled

```jsx live=true
<WithState>
  {(value, setValue) => (
    <Checkbox
      checked={!!value}
      onChange={e => setValue(e.currentTarget.checked)}
      label="Controlled checkbox"
      name="6"
    />
  )}
</WithState>
```
