# Visage

## Installation

```sh transpile=false
npm install @byteclaw/visage
# or
yarn add @byteclaw/visage
```

## Usage

In order to use Visage in your application, make sure you wrap your application to `<DesignSystem />` or `<ResponsiveDesignSystem />` and provide `theme`.

### Non Responsive

```jsx transpile=false
import { createNPointTheme, DesignSystem, Text } from '@byteclaw/visage';

const theme = createNPointTheme(/* see below on how to set up theme */);

// responsive design system
<DesignSystem theme={theme}>
  <Text>Hello world!</Text>
</DesignSystem>;
```

### Responsive

```jsx transpile=false
import {
  createNPointTheme,
  ResponsiveDesignSystem,
  Text,
} from '@byteclaw/visage';

const theme = createNPointTheme(/* see below on how to set up theme */);

// responsive design system
<ResponsiveDesignSystem theme={theme}>
  <Text>Hello world!</Text>
</ResponsiveDesignSystem>;
```

## N Point Theme

`createNPointTheme` is [N pixel based grid system](https://spec.fm/specifics/8-pt-grid) with [Modular scale](https://www.modularscale.com/) used for typography.

### Grid

The grid is defined by [N pixel based grid system](https://spec.fm/specifics/8-pt-grid).

The size of the grid is specified by `baselineGridSize: number`. This value is then used to calculate spacings.

### Typography

Typography is using [Modular scale](https://www.modularscale.com/).

To set up the typography use:

```ts transpile=false
{
  /**
   * Base font size can be specified as responsive if you use ScaleValue
   */
  baseFontSize: number | ScaleValue<number>;
  baseLineHeightRatio: number;
  fontScaleRatio: number;
  fontFamilies: {
    body: string;
    heading: string;
    [custom: string]: string;
  }
}
```

**Make sure you set up the custom font families in your document, visage is not responsible for importing custom fonts.**

### Color palette

In theme every color in a palette accepts `string` or `scale`.

#### Scale

Scale is an object that have `{ values: string[], offset: number }`.

Offset is used to calculate a shade of a color.

```jsx transpile=false
createNPointTheme({
  colors: {
    body: 'black', // string color
    neutral: { // scale color
      values: [
        'rgb(249, 249, 249)', // neutral.-5
        'rgb(244, 244, 244)',
        'rgb(238, 238, 238)',
        'rgb(232, 232, 232)', // neutral.-2
        'rgb(227, 227, 227)', // neutral.-1
        '#DDDDDD',
        'rgb(184, 184, 184)', // neutral.1
        'rgb(147, 147, 147)', // neutral.2
        'rgb(111, 111, 111)',
        'rgb(74, 74, 74)',
        'rgb(37, 37, 37)', // neutral.5
      ],
      offset: 5,
    },
  }
});

// neutral shade is a shade of the color at index defined by offset in the scale.
<Box styles={{ color: 'neutral' }}></Box>
// is same as
<Box styles={{ color: 'neutral.0' }}></Box>
// is same as
<Box styles={{ color: '#DDDDDD' }}></Box>

// to get lighter shade use negative step
<Box styles={{ color: 'neutral.-1' }}></Box>
// is same as
<Box styles={{ color: 'rgb(227, 227, 227)' }}></Box>

// to get darker shade use positive step
<Box styles={{ color: 'neutral.1' }}></Box>
// is same as
<Box styles={{ color: 'rgb(184, 184, 184)' }}></Box>

```
