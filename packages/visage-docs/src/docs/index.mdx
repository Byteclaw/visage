Visage is accessible design system for React applications with first class Typescript support.

It takes only 3 steps to start your first Visage powered application.

## Install

<Tabs>
  <Tab label="npm">
    <CodeBlock className="language-console" transpile={false}>
      npm install --save @byteclaw/visage
    </CodeBlock>
  </Tab>
  <Tab label="yarn">
    <CodeBlock className="language-console" transpile={false}>
      yarn add @byteclaw/visage
    </CodeBlock>
  </Tab>
</Tabs>
