---
title: 'Emotion style generator'
description: 'What is emotion style generator and how it works.'
---

Visage uses emotion style generator by default. It&apos;s created by `createEmotionStyleGenerator()`.

This style generator doesn&apos;t produce CSS media queries but uses the current breakpoint matched by `window.matchMedia` API.

https://codesandbox.io/s/emotion-style-generator-o11fd?fontsize=14&hidenavigation=1&module=%2Fsrc%2Findex.js&theme=light

<Message status="warning">
  This style generator is not recommended for application where you cannot
  detect device on during the server side rendering.
</Message>
