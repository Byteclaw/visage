# Spin button

**`SpinButton` is an input widget that allows a user to pick a value from a discrete set of values.**

For example, in an alarm clock app you can use `SpinButton` to allow users to set up seconds as a value from `0` to `59`.

## Import

```js transpile=false
import { SpinButton } from '@byteclaw/visage';
```

## Examples

### Default

```jsx live=true
<SpinButton
  defaultValue="March"
  values={[
    'January',
    'February',
    'March',
    'April',
    'May',
    'June',
    'July',
    'August',
    'September',
    'October',
    'November',
    'December',
  ]}
/>
```

### Controlled

```jsx live=true
() => {
  const [value, setValue] = useState('March');

  return (
    <SpinButton
      onChange={setValue}
      value={value}
      values={[
        'January',
        'February',
        'March',
        'April',
        'May',
        'June',
        'July',
        'August',
        'September',
        'October',
        'November',
        'December',
      ]}
    />
  );
};
```

### Disabled

```jsx live=true
<SpinButton disabled values={['a', 'b', 'c', 'd']} />
```

### Read only

```jsx live=true
<SpinButton readOnly values={['a', 'b', 'c', 'd']} />
```
