# Text input

TextInput component.

## Import

```js transpile=false
import { TextInput } from '@byteclaw/visage';
```

## Examples

### Default input

```jsx live=true
<TextInput placeholder="Placeholder text" />
```

### Default input with larger font size

```jsx live=true
<TextInput
  placeholder="Placeholder text"
  styles={{ fontSize: 1, lineHeight: 1 }}
/>
```

### Invalid input

```jsx live=true
<TextInput invalid placeholder="Placeholder text" />
```

### Disabled input

```jsx live=true
<TextInput
  disabled
  defaultValue="Disabled input"
  placeholder="Placeholder text"
/>
```

### ReadOnly input

```jsx live=true
<TextInput
  readOnly
  defaultValue="Read only input"
  placeholder="Placeholder text"
/>
```

### Input with prefix

```jsx live=true
<TextInput prefix={<div>A</div>} placeholder="Placeholder text" />
```

### Input with suffix

```jsx live=true
<TextInput suffix="B" placeholder="Placeholder text" />
```

### Input with prefix and suffix

```jsx live=true
<TextInput prefix="A" suffix="B" placeholder="Placeholder text" />
```
