# FormField

Form field is composite component that groups [Label](./label), input control and [InlineError](./inline-error) together.

Every `FormField` has `id` prop required.

## Import

```js transpile=false
import { FormField } from '@byteclaw/visage';
```

## Examples

### Form field with label

Default form field uses [TextInput](./text-input) as default input control.

```jsx live=true
<FormField id="form-field" label="Label" />
```

### Required form field with label

```jsx live=true
<FormField id="required-form-field" label="Label" required />
```

### Invalid form field with label

```jsx live=true
<FormField
  id="invalid-form-field"
  label="Label"
  error="This field is required"
/>
```

### Form field without visible label

If you don't want the label to be visible use `hiddenLabel` prop. Make sure you specify `label` prop so it can be used for assistive technology.

```jsx live=true
<FormField id="form-field-hidden-label" label="Label" hiddenLabel />
```

### Invalid form field without error message

To show input as invalid but without error message, specify `error` as `true`.

```jsx live=true
<FormField id="invalid-form-field-without-message" label="Label" error />
```
