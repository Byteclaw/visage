# Radio

**A component that can be used to choose only one option from a distinct set of multiple choices.**

`Radio` must be used in a group so you can always choose an option from possible values.

## Import

```js transpile=false
import { Radio } from '@byteclaw/visage';
```

## Examples

Provide `id` and `name`. Name should be same for radio group that controls the same value. The `id` must be unique per radio.

### Default

```jsx live=true
<Fragment>
  <Radio
    defaultChecked
    label="Turned On"
    id="state-on"
    value="on"
    name="state"
  />
  <Radio label="Turned Off" id="state-off" value="off" name="state" />
</Fragment>
```

### Hidden labels

```jsx live=true
<Fragment>
  <Radio
    defaultChecked
    hiddenLabel
    id="hidden-on"
    name="hide"

    value="on"
  />
  <Radio
    hiddenLabel
    id="hidden-ff"
    name="hide"
```

### Controlled

```jsx live=true
function Example() {
  const [value, setValue] = useState('on');

  return (
    <>
      <Radio
        checked={value === 'on'}
        label="Turned On"
        id="state-on"
        onChange={e => setValue(e.currentTarget.value)}
        value="on"
        name="state-controlled"
      />
      <Radio
        checked={value === 'off'}
        label="Turned Off"
        id="state-off"
        onChange={e => setValue(e.currentTarget.value)}
        value="off"
        name="state-controlled"
      />
    </>
  );
}
```
