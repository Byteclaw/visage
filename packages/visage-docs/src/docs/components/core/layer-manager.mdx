# LayerManager

LayerManager is responsible for managing zIndexes accross your application. Every new layer must be wrapped in LayerManager in order to guarantee that zIndexes are correct.

Layer manager exposes hook [useLayerManager](/hooks/use-layer-manager) which can be used to get the current zIndex and nesting level information from LayerManager.

## Import

```js transpile=false
import { LayerManager } from '@byteclaw/visage';
```

## Usage

```js transpile=false
function Dialog() {
  const { zIndex } = useLayerManager();

  return <Box styles={{ position: 'absolute', zIndex }}>This is new layer</Box>;
}

<LayerManager>
  <Dialog />
</LayerManager>;
```

## Props

### Optional

#### increaseBy

- **number**

Telss the layer manager about the number that is added to current zIndex if nesting level increases.

```js transpile=false
<LayerManager incraseBy={100}>
  zIndex will be 200 because default zIndex from root is 100
  <LayerManager>
    zIndex will be 300 (because increaseBy is taken from context)
  </LayerManager>
</LayerManager>
```
