# Responsive design system

`ResponsiveDesignSystem` acts as a shortcut for `<DesignSystem responsive />`. It accepts the same props as [DesignSystem](/components/core/design-system) component.

## Import

```js transpile=false
import { ResponsiveDesignSystem } from '@byteclaw/visage';
```

## Usage

```js transpile=false
<ResponsiveDesignSystem theme={theme}>
  <Heading>Hello world</Heading>
<ResponsiveDesignSystem>
```
