### Disabled events

`disableAutoFocusItem` props turns of auto-focusing the first item in the menu list.

```jsx live
() => {
  const anchorRef = useRef();
  const [open, setOpen] = useState(false);

  return (
    <>
      <Button
        aria-controls="buttons-menu"
        aria-haspopup="menu"
        aria-expanded={open}
        onClick={() => setOpen(!open)}
        ref={anchorRef}
      >
        {open ? 'Close' : 'Pop up'}
      </Button>
      <Menu
        disableAutoFocusItem
        anchor={anchorRef}
        id="buttons-menu"
        onClose={() => setOpen(false)}
        open={open}
      >
        <MenuItem>One</MenuItem>
        <MenuItem>Two</MenuItem>
        <MenuItem>Three</MenuItem>
        <Divider />
        <MenuItem>Button One</MenuItem>
        <MenuItem>Button Two</MenuItem>
        <MenuItem>Button Three</MenuItem>
      </Menu>
    </>
  );
};
```

### Limit menu height

Bu default `Menu` height is not limited and it&apos;ll try to take as much space to fit the content. You can limit that by providing styles to `Popover` using `popoverProps.styles`.

```jsx live
() => {
  const anchorRef = useRef();
  const [open, setOpen] = useState(false);

  return (
    <>
      <Button
        aria-controls="buttons-menu"
        aria-haspopup="menu"
        aria-expanded={open}
        onClick={() => setOpen(!open)}
        ref={anchorRef}
      >
        {open ? 'Close' : 'Pop up'}
      </Button>
      <Menu
        anchor={anchorRef}
        id="buttons-menu"
        onClose={() => setOpen(false)}
        open={open}
        popoverProps={{
          styles: {
            maxHeight: 150,
          },
        }}
      >
        <MenuItem>One</MenuItem>
        <MenuItem>Two</MenuItem>
        <MenuItem>Three</MenuItem>
        <Divider />
        <MenuItem>Button One</MenuItem>
        <MenuItem>Button Two</MenuItem>
        <MenuItem>Button Three</MenuItem>
      </Menu>
    </>
  );
};
```
