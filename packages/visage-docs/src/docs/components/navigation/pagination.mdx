# Pagination

## Import

```js transpile=false
import {
  NextPageButton,
  Pagination,
  PreviousPageButton,
} from '@byteclaw/visage';
```

## Examples

### Arrow pagination only

```jsx live=true
<Pagination>
  <PreviousPageButton />
  <NextPageButton />
</Pagination>
```

### Arrow pagination for just one page (disabled arrows)

```jsx live=true
<Pagination>
  <PreviousPageButton disabled />
  <NextPageButton disabled />
</Pagination>
```
