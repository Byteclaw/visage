# Pagination

## Import

```js transpile=false
import {
  Pagination,
  PaginationNextPageButton,
  PaginationPreviousPageButton,
} from '@byteclaw/visage';
```

## Examples

### Arrow pagination only

```jsx live=true
<Pagination>
  <PaginationPreviousPageButton />
  <PaginationNextPageButton />
</Pagination>
```

### Arrow pagination for just one page (disabled arrows)

```jsx live=true
<Pagination>
  <PaginationPreviousPageButton disabled />
  <PaginationNextPageButton disabled />
</Pagination>
```
