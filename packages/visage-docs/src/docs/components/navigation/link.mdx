# Link

**A component that acts as a hyperlink if `href` attribute is provided.**

## Import

```js transpile=false
import { Link } from '@byteclaw/visage';
```

## Examples

```jsx live=true
<Text>
  Text text <Link href="#">Link</Link> text text
</Text>
```
