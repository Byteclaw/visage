# Tooltip

**Popup element to display when element receives keyboard focus or mouse hovers over it.**

`Tooltip` automatically provides `onBlur`, `onFocus`, `onMouseEnter` and `onMouseLeave` handlers. It&apos;s up to you to use them correctly otherwise tooltip won&apos; show up.

## Import

```js transpile=false
import { Tooltip } from '@byteclaw/visage';
```

## Examples

### Default

```jsx live=true
<Tooltip content="1">
  <Button>With tooltip</Button>
</Tooltip>
```
