# Banner

**An accessible component that can be used to notify about something major in an application.**

`Banner` has similar meaning as [`Message`](/feedback/message) but is meant to be used as a main feedback component in your application.
Based on `status` prop it renders a component with `role` attribute set to `alert` or `status`.

## Import

```js transpile=false
import { Banner } from '@byteclaw/visage';
```

## Examples

### Default banner

```jsx live=true
<Banner onDismiss={() => {}} title="Default banner">
  <Text>Banner text</Text>
</Banner>
```

### Flat banner

```jsx live=true
<Banner flat onDismiss={() => {}} title="Default flat banner">
  <Text>Banner text</Text>
</Banner>
```

### Not dismissable Banner

Just omit the usage of `onDismiss` prop.

```jsx live=true
<Banner title="Default flat banner">
  <Text>Banner text</Text>
</Banner>
```

### Variants

All variants are based on `status`Â prop.

```jsx live=true
<Fragment>
  <Banner onDismiss={() => {}} title="Default banner">
    <Text>Banner text</Text>
  </Banner>
  <Banner onDismiss={() => {}} status="info" title="Info banner">
    <Text>Banner text</Text>
  </Banner>
  <Banner onDismiss={() => {}} status="success" title="Success banner">
    <Text>Banner text</Text>
  </Banner>
  <Banner onDismiss={() => {}} status="warning" title="Warning banner">
    <Text>Banner text</Text>
  </Banner>
  <Banner onDismiss={() => {}} status="critical" title="Critical banner">
    <Text>Banner text</Text>
  </Banner>
</Fragment>
```
