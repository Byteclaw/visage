# Box skeleton

**Skeleton component for [Box](./box) component.**

## Import

```js transpile=false
import { BoxSkeleton } from '@byteclaw/visage';
```

## Examples

### Default Box

Default box acts as a square (`aspectRation` is `1` by default.)

```jsx live=true
<BoxSkeleton />
```

### Custom aspect ratio

Use `aspectRatio` prop, e.g. `aspectRatio={16 / 9}`

```jsx live=true
<BoxSkeleton aspectRatio={16 / 9} />
```

### Custom dimensions

If you need specific dimensions, just use `styles` prop.

```jsx live=true
<BoxSkeleton styles={{ width: 50, height: 10 }} />
```
