# Accordion

**A component that acts as a vertically stacked set of headings that can be expanded to reveal a content.**

`Accordion` are commonly used to reduce the need to scroll when content with multiple sections is displayed on a single page.

## Import

```js transpile=false
import { Accordion, AccordionItem } from '@byteclaw/visage';
```

## Examples

### Default

By default `Accordion` expands the first item and it can&apos;t be changed unless you make it controlled by using `open` prop on items.

```jsx live=true
<Accordion>
  <AccordionItem title="Section 1">
    <Text>Section 1 content</Text>
  </AccordionItem>
  <AccordionItem title="Section 2">
    <Text>Section 2 content</Text>
  </AccordionItem>
  <AccordionItem title="Section 3">
    <Text>Section 3 content</Text>
  </AccordionItem>
  <AccordionItem title="Section 4">
    <Text>Section 4 content</Text>
  </AccordionItem>
</Accordion>
```

### Controlled

To make `Accordion` controlled, set `open` prop on active item and use `onOpen` prop on individual items to control the state.

```jsx live=true
() => {
  const [item, setItem] = useState(0);
  return (
    <Accordion>
      <AccordionItem open={item === 0} onOpen={setItem} title="Section 1">
        <Text>Section 1 content</Text>
      </AccordionItem>
      <AccordionItem open={item === 1} onOpen={setItem} title="Section 2">
        <Text>Section 2 content</Text>
      </AccordionItem>
      <AccordionItem open={item === 2} onOpen={setItem} title="Section 3">
        <Text>Section 3 content</Text>
      </AccordionItem>
      <AccordionItem open={item === 3} onOpen={setItem} title="Section 4">
        <Text>Section 4 content</Text>
      </AccordionItem>
    </Accordion>
  );
};
```
