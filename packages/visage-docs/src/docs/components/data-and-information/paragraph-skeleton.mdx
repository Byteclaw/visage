# Paragraph skeleton

**Skeleton component for Paragraph**

## Import

```js transpile=false
import { ParagraphSkeleton } from '@byteclaw/visage';
```

## Examples

### Default

```jsx live
<ParagraphSkeleton />
```

### Custom lines count

```jsx live
<ParagraphSkeleton lines={2} />
```

### Bigger font size

```jsx live
<ParagraphSkeleton styles={{ fontSize: 2, lineHeight: 2 }} />
```
