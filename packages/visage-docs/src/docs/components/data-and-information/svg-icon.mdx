# Svg icon

## Import

```js transpile=false
import { SvgIcon } from '@byteclaw/visage';
```

## Examples

### Default

Default SVG icon uses the body text color specified in theme and the size is inherited from font size.

```jsx live=true
<SvgIcon
  icon={props => (
    <svg width="1em" height="1em" viewBox="0 0 24 24" {...props}>
      <path
        fill="currentColor"
        d="M19 6.41L17.59 5 12 10.59 6.41 5 5 6.41 10.59 12 5 17.59 6.41 19 12 13.41 17.59 19 19 17.59 13.41 12z"
      />
      <path d="M0 0h24v24H0z" fill="none" />
    </svg>
  )}
/>
```

### Bigger icon (font size based)

```jsx live=true
<SvgIcon
  icon={props => (
    <svg width="1em" height="1em" viewBox="0 0 24 24" {...props}>
      <path
        fill="currentColor"
        d="M19 6.41L17.59 5 12 10.59 6.41 5 5 6.41 10.59 12 5 17.59 6.41 19 12 13.41 17.59 19 19 17.59 13.41 12z"
      />
      <path d="M0 0h24v24H0z" fill="none" />
    </svg>
  )}
  styles={{ fontSize: 1, lineHeight: 1 }}
/>
```

### Bigger icon (height same as line height)

```jsx live=true
<SvgIcon
  icon={props => (
    <svg width="1em" height="1em" viewBox="0 0 24 24" {...props}>
      <path
        fill="currentColor"
        d="M19 6.41L17.59 5 12 10.59 6.41 5 5 6.41 10.59 12 5 17.59 6.41 19 12 13.41 17.59 19 19 17.59 13.41 12z"
      />
      <path d="M0 0h24v24H0z" fill="none" />
    </svg>
  )}
  styles={{
    // basically the same as fontSize: 2, lineHeight: 2
    // except that the height of icon will be the same as lineHeight
    iconSize: 2,
  }}
/>
```

### Custom styles

```jsx live=true
<SvgIcon
  icon={props => (
    <svg width="1em" height="1em" viewBox="0 0 24 24" {...props}>
      <path
        fill="currentColor"
        d="M19 6.41L17.59 5 12 10.59 6.41 5 5 6.41 10.59 12 5 17.59 6.41 19 12 13.41 17.59 19 19 17.59 13.41 12z"
      />
      <path d="M0 0h24v24H0z" fill="none" />
    </svg>
  )}
  styles={{
    color: 'red',
    iconSize: 2,
  }}
/>
```
