# Divider

Divider is used to separate sections or menus.

## Import

```js transpile=false
import { Divider } from '@byteclaw/visage';
```

## Horizontal

### Default divider

```jsx live
<Divider />
```

### Labelled divider

```jsx live
<Divider label="OR" />
```

### With custom font size

```jsx live
<Divider label="OR" styles={{ fontSize: 1, lineHeight: 1 }} />
```

### Vertical

### Default divider

```jsx live
<Box styles={{ height: '200px' }}>
  <Divider vertical />
</Box>
```

### Labelled divider

```jsx live
<Box styles={{ height: '200px' }}>
  <Divider label="OR" vertical />
</Box>
```

### With custom font size

```jsx live
<Box styles={{ height: '200px' }}>
  <Divider label="OR" styles={{ fontSize: 1, lineHeight: 1 }} vertical />
</Box>
```
