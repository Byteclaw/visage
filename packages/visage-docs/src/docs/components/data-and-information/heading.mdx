# Heading

## Import

```js transpile=false
import { Heading } from '@byteclaw/visage';
```

## Examples

### Default (level 1)

```jsx live
<Heading>Heading</Heading>
```

### Other levels

Use `level` prop to handle different levels of heading

```jsx live
<Heading level={3}>Heading 3</Heading>
```
