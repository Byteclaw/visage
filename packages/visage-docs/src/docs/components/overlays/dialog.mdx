# Dialog

**A component that works as a window overlaid on either the primary window or another `Dialog`.**

`Dialog` works as a focus trap. That means that <kbd>Tab</kbd> or <kbd>Shift</kbd> + <kbd>Tab</kbd> do not move focus outside of the dialog.

`Dialog` also forbids a user to interact with a content outside of the dialog.

## Import

```js transpile=false
import { Dialog } from '@byteclaw/visage';
```

## Examples

```jsx live=true
function Example() {
  const [show, setShow] = useState(false);

  if (!show) {
    return (
      <Button onClick={() => setShow(true)} type="button">
        Show dialog
      </Button>
    );
  }

  return (
    <Dialog
      id="dialog-unique-id"
      label="Closable modal"
      onClose={e => {
        console.log(e);
        setShow(false);
      }}
    >
      {Array(50)
        .fill(null)
        .map((_, i) => (
          <form key={i} action="">
            <label htmlFor="searchable">Searchable select value</label>
            <Select
              onChange={value => console.log(value)}
              options={() => Promise.resolve(['a', 'b', 'c', 'd'])}
              placeholder="Select a value"
              searchable
            />
          </form>
        ))}
    </Dialog>
  );
}
```

## Keyboard Interaction

When a `Dialog` opens, focus is moved to an element defined by `focusElementRef` prop or if not provided but dialog is closable, it will be moved to Close button.

<DescriptionList>
  <DescriptionListItem
    term={<KeyboardKey>Tab</KeyboardKey>}
    description={
      <>
        <ul>
          <li>Moves focus to the next tabbable element inside the dialog.</li>
          <li>
            If focus is on the last tabbable element inside the dialog, moves
            focus to the first tabbable element inside the dialog.
          </li>
        </ul>
      </>
    }
  />
  <DescriptionListItem
    term={
      <Text styles={{ whiteSpace: 'nowrap' }}>
        <kbd>Shift</kbd> + <kbd>Tab</kbd>
      </Text>
    }
    description={
      <>
        <ul>
          <li>
            Moves focus to the previous tabbable element inside the dialog.
          </li>
          <li>
            If focus is on the first tabbable element inside the dialog, moves
            focus to the last tabbable element inside the dialog.
          </li>
        </ul>
      </>
    }
  />
  <DescriptionListItem
    term={<KeyboardKey>Escape</KeyboardKey>}
    description="If the dialog is closable, closes it."
  />
</DescriptionList>
