# Modal

**A component that works as a building element that is overlaid on either the primary window or another `Modal` based components.**

`Modal` component is used to create high order components as [`Dialog`](/components/overlays/dialog), [`Popover`](/components/overlays/popover). It provides close on click away, close on <kbd>Escape</kbd> key down, backdrop functionality and correct `zIndex` management.

`Modal` works as a focus trap. That means that <kbd>Tab</kbd> or <kbd>Shift</kbd> + <kbd>Tab</kbd> do not move focus outside of the modal.

`Modal` also can forbid a user to interact with a content outside of the modal if `backdrop` is enabled.

## Import

```js transpile=false
import { Modal } from '@byteclaw/visage';
```

## Examples

### Default

```jsx live=true
function Example() {
  const [show, setShow] = useState(false);

  if (!show) {
    return (
      <Button onClick={() => setShow(true)} type="button">
        Show modal
      </Button>
    );
  }

  return (
    <Modal
      id="unique-id"
      label="Closable modal"
      onClose={e => {
        console.log(e);
        setShow(false);
      }}
    >
      Content
    </Modal>
  );
}
```

## Keyboard Interaction

When a `Modal` opens, focus is moved to an element defined by `focusElementRef` prop.
