# Modal

## Import

```js transpile=false
import { Modal } from '@byteclaw/visage';
```

## Examples

```jsx live=true
<WithState defaultValue={false}>
  {(show, setShow) => {
    if (!show) {
      return (
        <Button onClick={() => setShow(true)} type="button">
          Show modal
        </Button>
      );
    }

    return (
      <Modal
        id="unique-id"
        label="Closable modal"
        onClose={e => {
          console.log(e);
          setShow(false);
        }}
      >
        Content
      </Modal>
    );
  }}
</WithState>
```
