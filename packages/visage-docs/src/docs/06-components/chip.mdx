**Compact elements that describe an input, attribute or action.**

## Import

```js transpile=false
import { Chip } from '@byteclaw/visage';
```

## Examples

### Default

Default `Chip` component is non interactive.

```jsx live=true
<Chip>Non interactive chip</Chip>
```

### Interactive

To make `Chip` interactive provide `onClick` handler.
Now `Chip` will be focusable and starts reacting to clicks, `Enter` and `Space` key events.

```jsx live=true
<Chip onClick={() => window.alert('Clicked')}>Interactive chip</Chip>
```

### Deletable

To make chip deletable, pass `onDelete` handler.

```jsx live=true
<Fragment>
  <Chip onDelete={() => window.alert('Deleted')}>
    Deletable non interactive chip
  </Chip>
  <Chip
    onClick={() => window.alert('Clicked')}
    onDelete={() => window.alert('Deleted')}
  >
    Deletable and clickable chip
  </Chip>
</Fragment>
```

### Small

Small `Chip` useful to tag things in headers.

```jsx live=true
<Fragment>
  <Flex
    styles={{
      '& > *': { m: 0.5 },
      backgroundColor: 'transparent',
      flexWrap: 'wrap',
    }}
  >
    <Chip small>Small</Chip>
    <Chip small variant="primary">
      Primary
    </Chip>
    <Chip small variant="success">
      Success
    </Chip>
    <Chip small variant="warning">
      Warning
    </Chip>
    <Chip small variant="danger">
      Danger
    </Chip>
    <Chip small outlined>
      Outlined
    </Chip>
    <Chip outlined small variant="primary">
      Primary
    </Chip>
    <Chip outlined small variant="success">
      Success
    </Chip>
    <Chip outlined small variant="warning">
      Warning
    </Chip>
    <Chip outlined small variant="danger">
      Danger
    </Chip>
    <Chip small outlined onDelete={() => window.alert('Deleted')}>
      Outlined removable
    </Chip>
  </Flex>
</Fragment>
```
