**A component that provides visual feedback for an action with an unknown state.**

This component is commonly used when a user action results in some action that doesn&apos;t have predetermined duration.

<Message status="warning">
  This component doesn&apos;t work in IE11 because it doesn&apos;t support SVG
  animations.
</Message>

## Import

```js transpile=false
import { Spinner } from '@byteclaw/visage';
```

## Examples

### Default

Spinner by default uses `currentColor` to set up stroke color and it's size is the same as current font size.

```jsx live=true
<Spinner />
```

### Custom size or color

```jsx live=true
<Spinner styles={{ color: 'blue', fontSize: 2, lineHeight: 2 }} />
```

### Use Spinner in Button

```jsx live=true
<Button>
  <Spinner styles={{ mr: 1 }} /> Button with Spinner
</Button>
```
