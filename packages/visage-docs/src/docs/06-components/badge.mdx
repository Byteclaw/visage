**Simple indicator component to display sum of stacked items, notify on events or display warnings next to icons.**

## Import

```js transpile=false
import { Badge } from '@byteclaw/visage';
```

## Examples

### Default

```jsx live=true
<Badge content="1">
  <Box styles={{ background: 'pink', width: '30px', height: '30px' }} />
</Badge>
```

## Faces

- `BadgeContainer`
- `BadgeContent`
