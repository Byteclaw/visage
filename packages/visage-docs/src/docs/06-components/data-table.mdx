**A component for tabular data.**

## Examples

### Simple Table

```jsx live=true
<Card styles={{ p: 0, width: 500 }}>
  <DataTable>
    <DataTableHeader>
      <DataTableHeaderRow>
        <DataTableHeaderColumn>Product</DataTableHeaderColumn>
        <DataTableHeaderColumn numeric>Quantity</DataTableHeaderColumn>
        <DataTableHeaderColumn numeric>Price</DataTableHeaderColumn>
      </DataTableHeaderRow>
    </DataTableHeader>
    <DataTableBody>
      <DataTableRow>
        <DataTableColumn>Product 1</DataTableColumn>
        <DataTableColumn numeric>1</DataTableColumn>
        <DataTableColumn numeric>10.00 €</DataTableColumn>
      </DataTableRow>
      <DataTableRow>
        <DataTableColumn>Product 2</DataTableColumn>
        <DataTableColumn numeric>2</DataTableColumn>
        <DataTableColumn numeric>10.00 €</DataTableColumn>
      </DataTableRow>
      <DataTableRow>
        <DataTableColumn>Product 3</DataTableColumn>
        <DataTableColumn numeric>2</DataTableColumn>
        <DataTableColumn numeric>10.00 €</DataTableColumn>
      </DataTableRow>
    </DataTableBody>
    <DataTableFooter>
      <DataTableFooterRow>
        <DataTableHeaderColumn total>Totals</DataTableHeaderColumn>
        <DataTableColumn numeric total>
          5
        </DataTableColumn>
        <DataTableColumn numeric total>
          50.00 €
        </DataTableColumn>
      </DataTableFooterRow>
    </DataTableFooter>
  </DataTable>
</Card>
```
