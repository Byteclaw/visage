**An accessible component that can be used to notify about something minor in an application.**

`Message` has similar meaning as [`Banner`](/feedback/banner) but with more compact look and suited to communicate multiple messages to a user. Based on `status` prop it renders a component with `role` attribute set to `alert` or `status`.

## Import

```js transpile=false
import { Message } from '@byteclaw/visage';
```

## Examples

### Default

```jsx live=true
<>
  <Message>Hello world!</Message>
  <Message inverted>Hello inverted world!</Message>
</>
```

### Dismissable

To make message dismissable, pass `onDismiss` handler. You can also specify a custom dismiss button.

```jsx live=true
<Fragment>
  <Message onDismiss={() => window.alert('Dismissd')}>
    Dismissable message
  </Message>
  <Message
    status="success"
    renderDismiss={props => (
      <Button variant="primary" styles={{ ml: 'auto' }} {...props}>
        Dismiss
      </Button>
    )}
    onDismiss={() => window.alert('Dismissed')}
  >
    Dismissable message
  </Message>
</Fragment>
```

### Variants

All variants are based on `status` prop and optional `inverted` prop.

```jsx live=true
<>
  <Message>Default</Message>
  <Message status="danger">Danger</Message>
  <Message status="info">Info</Message>
  <Message status="success">Success</Message>
  <Message status="warning">Warning</Message>
  <Message inverted>Default</Message>
  <Message inverted status="danger">
    Danger
  </Message>
  <Message inverted status="info">
    Info
  </Message>
  <Message inverted status="success">
    Success
  </Message>
  <Message inverted status="warning">
    Warning
  </Message>
</>
```
