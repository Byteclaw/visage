**Label component for form inputs.**

## Import

```js transpile=false
import { Label } from '@byteclaw/visage';
```

## Examples

### Basic field

```jsx live=true
<Fragment>
  <Label htmlFor="field-1">Field Label</Label>
  <TextInput id="field-1"></TextInput>
</Fragment>
```

### Field for required input

```jsx live=true
<Fragment>
  <Label htmlFor="field-2" required styles={{ fontSize: 2, lineHeight: 2 }}>
    Field label
  </Label>
  <TextInput id="field-2" required></TextInput>
</Fragment>
```

### Hidden field

```jsx live=true
<Fragment>
  <Label htmlFor="field-3" hidden>
    Field label
  </Label>
  <TextInput id="field-3"></TextInput>
</Fragment>
```
