**A component for vertical indexes of text or images.**

## Import

```js transpile=false
import { List, ListItem } from '@byteclaw/visage';
```

## Examples

```jsx live=true
<Fragment>
  <List>
    <ListItem>One</ListItem>
    <ListItem active>Two (active)</ListItem>
    <ListItem>Three</ListItem>
  </List>
  <Divider />
  <List>
    <ListItem role="button" tabIndex={0}>
      Button One
    </ListItem>
    <ListItem role="button" tabIndex={0}>
      Button Two
    </ListItem>
    <ListItem role="button" tabIndex={0}>
      Button Three
    </ListItem>
  </List>
</Fragment>
```

- `ListItemsContainer`
