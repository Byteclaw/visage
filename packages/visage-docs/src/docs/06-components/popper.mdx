**A component that renders content near an element based on `placement` rules.**

`Popper` can be used to implement tooltips or any extra element that pops out near some element.

Unlinke [`Popover`](/components/overlays/popover), `Popper` doesn&apos;t block scroll.

## Import

```js transpile=false
import { Popper } from '@byteclaw/visage';
```

## Usage

`Popper` doesn&apos;t have any default placement set. To set it up use `placement` prop that must be a responsive value which accepts a prioritized array of preffered placements. Any placement must have it&apos;s transform origin set up to say where the anchor should be positioned if given placement is considered for use.

```jsx live=true
() => {
  const anchorRef = useRef();
  const [open, setOpen] = useState(false);

  return (
    <Fragment>
      <Button onClick={() => setOpen(!open)} ref={anchorRef} type="button">
        {open ? 'Hide' : 'Show'} popper
      </Button>
      <Popper
        anchor={anchorRef}
        open={open}
        placement={[
          [
            {
              placement: Placement.topLeft,
              horizontal: 'left',
              vertical: 'bottom',
            },
          ],
        ]}
      >
        <Flex
          styles={{
            alignItems: 'center',
            backgroundColor:
              'color(shades if(isDark color(shades tint(10%)) color(shades shade(10%))))',
            height: 500,
            width: '100%',
          }}
        >
          Content
        </Flex>
      </Popper>
    </Fragment>
  );
};
```

### Inside scrollable container

Sometimes you have your anchor element inside a scrollable container. In this case you must use `scrollContainerRef` to properly position popper against anchor.

```jsx live=true
() => {
  const anchorRef = useRef();
  const [open, setOpen] = useState(false);

  return (
    <Box styles={{ height: 500, overflowY: 'auto' }}>
      <Box styles={{ my: '400px' }}>
        <Button onClick={() => setOpen(!open)} ref={anchorRef} type="button">
          {open ? 'Hide' : 'Show'} popper
        </Button>
        <Popper
          anchor={anchorRef}
          open={open}
          placement={[
            [
              {
                placement: Placement.topLeft,
                horizontal: 'left',
                vertical: 'bottom',
              },
            ],
          ]}
        >
          <Flex
            styles={{
              alignItems: 'center',
              backgroundColor:
                'color(shades if(isDark color(shades tint(10%)) color(shades shade(10%))))',
              height: 500,
              width: '100%',
            }}
          >
            Content
          </Flex>
        </Popper>
      </Box>
    </Box>
  );
};
```
