**Accessible Popup element to display when an element receives keyboard focus or mouse hovers over it.**

`Tooltip` automatically provides `onBlur`, `onFocus`, `onMouseEnter` and `onMouseLeave` handlers and `aria-describedby` props to identify which element describes an element. It&apos;s up to you to use them correctly otherwise tooltip won&apos;t show up or fulfill accessibility requirements.

## Import

```js transpile=false
import { Tooltip } from '@byteclaw/visage';
```

## Examples

### Default

```jsx live=true
<Tooltip content="1">
  <Button>With tooltip</Button>
</Tooltip>
```

### Keyboard Interaction

<DescriptionList>
