# useLayerManager

This hook is just a helper to access current nesting level and zIndex of [LayerManager](/components/layer-manager).

## Import

```js transpile=false
import { useLayerManager } from '@byteclaw/visage';
```

## Usage

```jsx
function MyComponent() {
  const { level, zIndex } = useLayerManager();

  return (
    <Box>
      Current nesting level {level} and zIndex {zIndex}
    </Box>
  );
}
```

## Returns

This hook returns an object with the following properties:

- **level** - current layer's nesting level
- **zIndex** - current layer's zIndex
- **increaseBy** - current increaseBy setting.
