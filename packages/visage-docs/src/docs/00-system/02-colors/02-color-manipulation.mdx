To ease up working with colors during development Visage offers `color()` function and a range of modificators that can be used to manipulate colors.

This way you can manipulate colors in different ways and produce interesting results. For example you can generate an accessible font color for different background colors.

Try different combinations in the playground.

<ColorPlayground color="color(white contrast())" />

<Message status="info">
  You can chain color modificators where each modificator will receive a color
  produced by the previous one.
</Message>

## Color function

`color()` function is used as a value for [color CSS attributes](#available-color-css-attributes). It takes a color as the first argument and any number of modificators. Color can be a safe color name, a color name from the color palette, hex code, `rgb()/rgba()` or `hsl()/hsla()`.
